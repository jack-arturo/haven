# Haven on Etsy - Setup & Strategy Guide

## Overview

Etsy provides additional distribution channel for Haven products alongside your main website. This guide walks you through setting up an Etsy shop and maintaining consistency across both platforms.

## Why Etsy?

**Advantages:**
- Built-in audience of buyers searching for handmade goods
- 5-6% of Haven website traffic likely comes from Etsy awareness
- Lower risk than relying solely on direct website
- Etsy handles payment processing (you receive funds after Etsy fees)
- Built-in review system builds credibility

**Disadvantages:**
- Etsy takes ~5% + payment processing fees
- Less brand control (Etsy branding prominent)
- Algorithm changes affect visibility
- Must maintain shop reputation

## Step 1: Create Etsy Shop

1. Go to **etsy.com**
2. Sign up or log in
3. Click "Sell on Etsy"
4. Choose shop name: `HavenCoversCo` or `HavenMoroccan`
5. Complete shop setup:
   - Shop location
   - Shop policies
   - Payment method (where Etsy sends earnings)
   - Billing address

### Shop Name Recommendations
- `HavenCoversCo`
- `HavenCollection`
- `HavenMoroccan`
- `HavenTextiles`
- Avoid overly generic names

## Step 9: Manage Inventory

### Inventory Tracking

**Goal:** Keep inventory consistent between platforms

**Method:**
1. Use spreadsheet to track:
   - Product name
   - Size
   - Quantity available
   - Units sold (Etsy + Website)
   - Remaining stock

2. Example:
```
Product: Beni Ourain
Size 65": 50 units - 3 sold (2 Etsy, 1 website) = 47 remaining
Size 75": 50 units - 1 sold (Etsy) = 49 remaining
Size 85": 50 units - 0 sold = 50 remaining
```

3. Update Etsy quantity when stock changes
4. Pause listing if out of stock
5. Restock before relaunching

### Stock Management Options

**Option A: Manual tracking**
- Update Etsy and website as you make sales
- Spreadsheet-based
- Reliable but time-consuming

**Option B: Etsy's inventory sync**
- Some Etsy apps sync with external inventory
- More complex but automated
- Look into Etsy apps that offer this

**Option C: Separate inventory**
- Keep some inventory for website only
- Some for Etsy only
- More complex to manage

**Recommendation:** Start with Option A (manual), scale to Option B later.

## Etsy Advertising (Optional)

### Etsy Ads
- Auto-promoted listings (starts at $0.20/day)
- Only pay for clicks
- Good for new shops (low budget to start)
- Start with $5-10/day budget

### Pros
- Increases visibility
- Etsy handles targeting
- Low barrier to entry
- Easy to scale

### Cons
- Reduces profit margin
- Results vary
- Need to monitor ROI

### Recommendation
- Start organic (no ads) for first month
- Monitor performance
- Add Etsy Ads if sales plateau
- Budget $50-100/month for ads

## Troubleshooting Etsy Issues

### Low views
- Review SEO (keywords, tags)
- Check competitor listings
- Consider Etsy Ads
- Improve product photos
- Refresh listings periodically

### High views, low sales
- Check pricing vs competitors
- Review product descriptions
- Improve photos
- Check reviews/ratings
- Add FAQs to address concerns

### Shipping complaints
- Review shipping speed
- Update expected delivery times
- Consider upgrade shipping method
- Improve packaging

### Negative reviews
- Respond professionally
- Offer solutions (refund, replacement)
- Learn from feedback
- Make improvements
- Request review removal if warranted
